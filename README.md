# 🛒 Cadastro de Produtos em Python

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de criar um sistema simples de gerenciamento de produtos utilizando arquivos de texto. Ele permite cadastrar, listar e excluir produtos de forma prática através do terminal.

## 📄 Descrição

O sistema funciona por meio de um menu interativo com as seguintes opções:

* **1 - Cadastrar produto:**
  Permite inserir código, nome, preço e quantidade do produto, salvando essas informações no arquivo `produtos.txt`.

* **2 - Listar produtos:**
  Exibe todos os produtos cadastrados, mostrando seus dados de forma organizada.

* **3 - Excluir produto:**
  Permite remover um produto com base na linha informada pelo usuário.

* **4 - Sair:**
  Encerra o programa.

Os dados são armazenados no seguinte formato:

```
codigo|nome|preco|quantidade
```

O projeto utiliza conceitos importantes da programação em Python, como:

* Estrutura de repetição (`while`)
* Estruturas condicionais (`if`, `elif`)
* Manipulação de arquivos (`open`, `read`, `write`)
* Listas (para remoção de dados)
* Entrada de dados (`input`)
* Biblioteca `os` para limpeza do terminal
